# Student Management System

Student Management System is a web-based application developed using Java, Servlets, JSP, and MySQL.  
It helps administrators manage student records efficiently through a simple interface.

## Features
- Add Student
- View Students
- Update Student Details
- Delete Student

## Technologies Used
- Java
- Servlet
- JSP
- JDBC
- MySQL
- Apache Tomcat
- Eclipse IDE

## How to Run
1. Import the project into Eclipse.
2. Configure Apache Tomcat Server.
3. Setup MySQL database.
4. Run the project on the server.
